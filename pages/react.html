React JS



This page demonstrates my knowledge using ReactJS. There were many exercises I did but these two mentioned below were the most challenging projects I worked on using React. For each project we used different APIs to get our information. Scroll through and check them out!

Ravenous

This was the project I worked on while learning new React concepts and applying it along the way. In this project, I built a website called "Ravenous", a Yelp-like web application. The application uses Yelp API to search for businesses and displayes it below. The user can search for a business within a city and the application will display its ratings and address.

Jammming

This was the capstone project marking the end of the course. In this project, I built a React web application called Jammming. I used my knowledge of React components, state, and requests with the Spotify API to build a website that allows users to search the Spotify library, create a custom playlist, then save it to their Spotify account. The user can choose can search with either name of the song, Artist or Album.
